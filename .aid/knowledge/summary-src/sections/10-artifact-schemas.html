Artifact Schemas

The structural schema of every artifact the AID methodology produces — STATE files, SPEC/REQUIREMENTS/feature/task files, settings, and the three manifest formats — with required fields, closed enums, producers, consumers, and validation points.

Task type enum (8 values, closed)

RESEARCH, DESIGN, IMPLEMENT, TEST, DOCUMENT, MIGRATE, REFACTOR, CONFIGURE. The type drives how the executor agent works and how the reviewer evaluates. No other values are valid.
