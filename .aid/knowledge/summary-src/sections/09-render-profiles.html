Render Profiles

Each supported host tool is encoded as one profiles/<tool>.toml profile consumed by run_generator.py. All five profiles share a uniform {agents,skills,aid} install layout under their tool root. The profile pins the tool root, the root context file, the agent format (markdown for four tools; toml for Codex, dormant), and any tool-name remaps. One card per profile below. (per architecture.md, integration-map.md, schemas.md §9)
