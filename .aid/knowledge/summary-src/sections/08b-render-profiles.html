Build & Install

Render Profiles

Each supported host tool is encoded as one profiles/<tool>.toml profile consumed by run_generator.py. A profile pins the install root, the per-tool agent format, the root context file, and any tool-specific layout (split roots, rule files, tool-name remaps). One card per profile below — every value is the profile's verified output_root, project_context_file, and [agent].format. (per architecture.md, integration-map.md, pipeline-contracts.md)

Reference / dogfood layout

Claude Code

Anthropic Claude Code — the canonical reference target AID itself dogfoods.

Install root: .claude/ (agents, skills, templates, recipes, scripts).
Context file: root CLAUDE.md.
Agent format: markdown (YAML frontmatter). No tool-name remap — abstract names pass through. (per profiles/claude-code.toml)

Split-root layout

Codex CLI

OpenAI Codex CLI — the only profile with a two-root split layout.

Install root: .codex/ (agents) + .agents/ (skills, templates).
Context file: root AGENTS.md.
Agent format: toml — agents are TOML files; tools declared inline in developer_instructions prose, not a tool list. (per profiles/codex.toml)

IDE + .mdc rules

Cursor

Cursor IDE — adds always-on .mdc methodology rules alongside skills.

Install root: .cursor/ (rules via [[extras.rules]] .mdc, skills, agents, templates).
Context file: root AGENTS.md.
Agent format: markdown. Tool remap: Bash → Terminal (the only non-identity tool map across all profiles). (per profiles/cursor.toml)

Native Agent Skills · no MCP

GitHub Copilot CLI

GitHub Copilot CLI — native Agent Skills under the .github/ tree.

Install root: .github/ (agents/*.agent.md, native skills/<slug>/SKILL.md, scripts, templates, recipes).
Context file: root AGENTS.md.
Agent format: copilot-agent. Tool remap: Bash → shell. No MCP servers emitted. (per profiles/copilot-cli.toml)

Trigger-frontmatter rules

Google Antigravity

Google Antigravity (Windsurf lineage) — sub-agents reshape into .md rule files with trigger: frontmatter.

Install root: .agent/ (rules/*.md with trigger: frontmatter, native skills/<slug>/SKILL.md, scripts, templates, recipes).
Context file: root AGENTS.md.
Agent format: antigravity-rule. Methodology rules emit via RuleEntry.output_filename as .md (not .mdc). (per profiles/antigravity.toml)
