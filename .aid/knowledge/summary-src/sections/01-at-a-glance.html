Polyglot Toolchain

Markdown, Bash (177 .sh files), Python 3.11+ (stdlib only), JavaScript/ESM, PowerShell 5.1+, CSS3, HTML5, TOML, YAML. No application frameworks — single runtime library is Mermaid, fetched dynamically. (per technology-stack.md, project-structure.md)

Architecture

Single-Source Compilation

One canonical source (canonical/) is rendered by run_generator.py into five byte-identical install trees — the enforced render invariant spans 6 trees (canonical/ + 5 profiles). Never edit the profile trees directly — edit canonical/ and re-run the generator. (A 7th on-disk copy, the repo's own hand-maintained dogfood .claude/, sits outside this render set — internal tooling, see §14.) (per architecture.md, project-structure.md)

No Application Code

Methodology Distribution

This repo ships no application — it ships documentation rendered into install bundles. No Docker, no cloud, no database. Quality gates are the AID skills themselves running on the maintainer's local workstation. (per infrastructure.md)
