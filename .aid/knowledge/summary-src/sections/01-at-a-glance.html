Polyglot Toolchain

Architecture

Single-Source Compilation

One canonical source (canonical/) is rendered by run_generator.py into three byte-identical install trees. Never edit the profile trees directly — edit canonical/ and re-run the generator. (per architecture.md)

No Application Code

Methodology Distribution

This repo ships no application — it ships documentation rendered into install bundles. No Docker, no cloud, no database. Quality gates are the AID skills themselves running on the maintainer's local workstation. (per infrastructure.md)
