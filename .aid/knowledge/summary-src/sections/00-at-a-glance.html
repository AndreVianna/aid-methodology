Overview

What is AID?

AID installs a structured AI workflow into any repository. It tells AI agents how to discover a project, gather requirements, write specs, plan work, execute tasks, and ship releases — one gated phase at a time, with a human in the loop at every step.

The Problem

AI without structure ships hallucination

Dropping an AI agent into a codebase without a Knowledge Base produces technically plausible but architecturally wrong code. AID solves this with Discovery-first: build the KB before writing a line of spec.

The Solution

Waterfall discipline at AI speed

AID revives Waterfall's sequential rigour — understand, specify, plan, build, verify, ship — but executes each phase with AI agents so the economics flip: discovery takes hours, not months.

The Delivery

A CLI installer for 5 host tools

AID ships as a multi-profile CLI installer. Install into a repo once; get 14 skills, 9 agents, and the full pipeline ready inside Claude Code, Codex, Cursor, GitHub Copilot CLI, or Antigravity.

AID 6-phase pipeline: Discover → Describe / Define → Specify → Plan → Detail → Execute. Every phase transition requires human approval. (aid-config bootstraps before the pipeline; Deploy and Monitor are optional Deliver skills after it.)
