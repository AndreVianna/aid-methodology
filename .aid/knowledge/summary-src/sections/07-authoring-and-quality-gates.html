Severity scale (5 levels)

Finding tags

Findings carry bracketed severity tags in the order [CRITICAL], [HIGH], [MEDIUM], [LOW], [MINOR]. The worst severity dominates the grade; count modifies it. Tags are also used by build-metrics.sh to tally debt items. (per coding-standards.md §8d)

Grade computation

grade.sh — deterministic

canonical/scripts/grade.sh (266 lines) converts an issue list with severity tags into a letter grade (A+ through F). The --non-functional flag forces F. The grade is computed by the script, never by the reviewing agent, making it tamper-resistant. (per module-map.md §4f)
