Generator

run_generator.py

87-line orchestrator. Loops all 5 profiles/*.toml, calls render_agents, render_skills, render_templates, render_canonical_scripts, render_recipes per profile, performs deletion pass via emission manifest diff, then runs VERIFY (deterministic) (hard) and VERIFY (advisory). Python 3.11+ stdlib only. (per module-map.md §3, infrastructure.md)

End-user install

setup.sh / setup.ps1

Interactive multi-select menu — [1] Claude Code [2] Codex [3] Cursor [4] GitHub Copilot CLI [5] Antigravity [6] Done. Copies each selected profile tree into the user's project root. Existing identical files are skipped; differing files prompt unless --force is passed. Bash 4+ (Linux/macOS/Git Bash) or PowerShell 5.1+ (Windows). (per infrastructure.md)

4 agent formats / split roots

Per-tool layouts

render_agents emits one of 4 formats per [agent].format: markdown (Claude Code, Cursor), toml (Codex split layout — .codex/agents/ + .agents/), copilot-agent (Copilot CLI native Agent Skills under .github/), and antigravity-rule (Antigravity .agent/rules reshape). Cursor + Copilot CLI also remap the Bash tool name (Terminal / shell); MCP-style tools are omitted on hosts that lack them. (per architecture.md, schemas.md §7, §9)

Option-A AGENTS.md collision handler

Four of the five tools — Codex, Cursor, Copilot CLI, Antigravity — write a root AGENTS.md (Claude Code uses CLAUDE.md only). When ≥2 AGENTS.md-writing tools are selected, setup.sh sets AGENTS_COLLISION=1, warns once, and resolves non-interactively with last-installed-wins — the survivor is the highest-numbered selected tool's install block (fixed per-tool order). The handler is gated: it only fires when AGENTS_COLLISION=1 and the destination basename is AGENTS.md, so a single-writer install never triggers auto-overwrite. (per infrastructure.md §Install Pipeline, architecture.md §Install-time data flow)

Never edit profile trees directly

Files under profiles/{claude-code,codex,cursor,copilot-cli,antigravity}/ are generator output. Edits are silently overwritten on the next python .claude/skills/aid-generate/scripts/run_generator.py run. Always edit canonical/ and re-run the generator. The emission manifest is the safety boundary — only previously-emitted files are eligible for deletion. (per coding-standards.md §7a)
