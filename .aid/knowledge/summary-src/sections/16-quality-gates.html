Quality Gates

AID's review-and-grade gates: the deterministic A-grade gating, the reviewer ledger, per-phase REVIEW→FIX loops, minimum-grade thresholds, and the discover review panel. Distinct from the automated test suites in test-landscape.md.

Two-Layer Quality Model

Layer 1

Automated Tests

82 canonical suites via tests/run-all.sh; CI-gated; catches functional regressions and format violations. Runs before quality gates.

Layer 2

Quality Gates (reviewed + graded)

A separate reviewer agent reads the output, classifies issues by [SEVERITY], and grade.sh computes the letter grade. Catches spec, architecture, and convention issues tests cannot detect.
