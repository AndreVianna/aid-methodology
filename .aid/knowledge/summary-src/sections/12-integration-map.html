Integration Map

AID is a distributed toolkit, not a networked service — most "integrations" are developer-side tools (git, gh, npm, PyPI, GitHub Actions) and the five host AI harnesses it installs into. The only runtime HTTP surface is the loopback-bound dashboard server.

GitHub Actions CI/CD 4 workflows: test.yml (master-only), installer-tests.yml (feature branches), release.yml (tags), docs.yml (master-only) Heavy gates (canonical suite + docs build) only run on master; feature branches skip them

Web page reviews: Playwright required

Any review of web output (including kb.html) MUST use Playwright to render the page and visually validate the result. Reading HTML source is insufficient. A review without Playwright visual validation is an automatic FAIL.
