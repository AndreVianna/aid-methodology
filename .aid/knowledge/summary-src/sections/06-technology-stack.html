Technology Stack

AID is polyglot by design: the same functionality must install through both Bash and PowerShell hosts, and ship via both npm and PyPI. The CLI and dashboard declare zero runtime dependencies.

PowerShell 5.1 gotchas

Four constructs that work in pwsh 7 but break on 5.1: TLS 1.2 not default, 3-argument Join-Path, -Encoding utf8NoBOM, $IsWindows, non-ASCII in no-BOM .ps1 files. All shipped .ps1 files must be ASCII-only.
