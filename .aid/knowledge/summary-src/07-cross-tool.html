7. Cross-Tool Delivery — One Pipeline, Three Install Bundles

AID ships the same methodology to three host AI coding tools. One canonical source lives at canonical/; python run_generator.py renders it into three profile trees under profiles/{claude-code,codex,cursor}/. Skill bodies are byte-identical across all 4 trees (canonical + 3 profiles), verified by .claude/skills/aid-generate/scripts/verify_deterministic.py. Per-host adaptations (file extensions, agent frontmatter format, deployment paths) live in per-profile TOMLs read by the generator. This replaces the pre-work-002 "triplication rule" + manual cross-tree sync model.
