## What is AID?

AID (Agentic Iterative Development) is a methodology for working effectively with AI coding
assistants. It provides structure, repeatability, and quality gates for teams building software
with AI pair-programmers.

- **Structured** — clear phases, roles, and artifacts
- **Agnostic** — works with any host tool (Claude, Copilot, Cursor, and more)
- **Lightweight** — minimal process overhead; adapts from solo to team scale
